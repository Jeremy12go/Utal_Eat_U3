## Instalación y ejecución (con Docker)

### 1. Clona el proyecto

```bash
    git clone https://github.com/tuUsuario/Utal_Eat_U3.git
    cd Utal_Eat_U3
```

### 2. Ejecuta el proyecto
```bash
    docker-compose up --build
```
