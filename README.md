## Instalación y ejecución (con Docker)

### 1. Clona el proyecto

```bash
    git clone https://github.com/tuusuario/SmartHome360.git
    cd SmartHome360
```

### 2. Ejecuta el proyecto
```bash
    docker-compose up --build
```
